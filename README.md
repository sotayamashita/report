# report

## Description

Generate report for daily with Golang

## Feature

* [ ] [Toggl](https://www.toggl.com/)

## Usage

```bash
$ report toggl
```

## Configuration

please create `config.tml`:

```
[toggl]
api_token = "<your api token>"
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/Sota Yamashita/report
```
